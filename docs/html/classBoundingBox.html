Detailed Description

A cuboid in 3D space for estimating the position of objects.

BoundedVolume objects have to be contained in a bounding box.

Note
Note, that a bounding box cannot be rendered

◆ intersection_interval()

Interval BoundingBox::intersection_interval ( const Ray &  ray) const

Returns which part of ray lies inside the box.

If the ray is given as gamma(t)=b+t*v where b is the base vector and v is the direction, this function will return the interval that contains all t such that gamma(t) lies in the cuboid.

Parameters
rayThe ray intersecting the box
Returns
The interval of arguments t for which ray.readBase()+ray.readDirection()*t is inside the box.
